## 🚀 How to Get Started

1. Clone the repository:
   ```bash
   git clone https://github.com/rafael-a-g-n/clips-demo.git
   ```
2. Navigate to project directory:
   ```bash
   cd clips-demo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the devolopment server:
   ```bash
   npm start
   ```
